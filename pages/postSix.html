Position: Relative

One way to modify the default position of an element is by setting its position property to relative This value allows you to position an element relative to its default static position on the web page.

Position: Absolute

Another way of modifying the position of an element is by setting its position to absolute. The element will be positioned reletive to its closest ancestor that has its position property set to anything other than static (e.g., relative, absolute, or fixed ), and offset properties can be used to adjust its final position from that reference point.
